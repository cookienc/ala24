## 4. 구현 기능
1. 회원 관련
    - 등록
    - 조회
    - 계좌에 돈 충전
    - 로그인 및 로그아웃
2. 상품 관련
    - 등록
    - 수정
    - 조회
3. 주문 관련
    - 주문
    - 주문 취소
    - 조회
4. 기타
    - 회원 및 회원의 권한 별 화면 구성
    - 페이징
    - 검색 및 정렬

## 5. 배운점 및 아쉬운점
- 배운점
  - 각각의 기능들이 유기적으로 연결하는 방법
  - 오류가 발생했을 때 대처법
- 아쉬운점
  - 기능 구현이 덜 된 점 ex) 장바구니
  - 화면 구성의 아쉬움
